# Ruby Media

Professional Web Development Company Website

## Features

- **Multi-language Support**: English (default), 中文, Bahasa Melayu, தமிழ்
- **Responsive Design**: Works on desktop, tablet, and mobile
- **WhatsApp Integration**: One-click contact button (+65 8237 6302)
- **Modern UI**: Clean design with smooth animations

## Project Structure

```
website-build/
├── index.html      # Main page
├── css/
│   └── style.css   # Styles
├── js/
│   └── main.js     # Interactions & i18n
└── assets/
    └── images/     # Static assets
```

## Local Preview

```bash
cd ~/Desktop/website-build
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript
- LocalStorage for language preference
